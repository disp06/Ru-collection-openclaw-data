## О проекте
- **Источники:** GitHub Search (ключевые слова: openclaw, openclaw-agent, openclaw-ru, русский и др.)
- **Фильтрация:** RU-ориентированность (русский язык, русские символы в описании/README)
- **Безопасность:** базовый анализ (скрытые секреты, подозрительные команды)

## Как добавить своё решение
1. Открой репозиторий на GitHub.
2. Добавь ключевое слово `openclaw` в описание или README.
3. Убедись, что есть русскоязычная документация.
