- **Источники:** GitHub Search (ключевые слова: openclaw, agent, mcp, integrations, russian и др.)

## Как добавить своё решение
1. Открой репозиторий на GitHub.
2. Добавь ключевое слово `openclaw` в описание или README.
3. Убедись, что есть русскоязычная документация и код.

Сборщик автоматически подхватит его в следующую итерацию.
